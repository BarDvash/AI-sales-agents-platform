# Commit and Push Changes

Commit all changes and push to remote repository.

## Steps

1. **Follow /commit-locally workflow:**
   - Analyze changes (git status, git diff)
   - Read README.md and decide if update needed
   - Update README if warranted
   - Stage all changes
   - Commit with `[AISalesAgentsPlatform]` prefix and detailed message

2. **Push to remote:**
   - Run `git push`
